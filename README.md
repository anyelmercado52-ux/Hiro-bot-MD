🤖 hiro BOT MD

«Bot de WhatsApp Multi-Dispositivo avanzado construido con Baileys 🚀»

---

🌐 Descripción

hiro Bot MD es un bot moderno de WhatsApp que utiliza la librería Baileys para funcionar en modo multi-dispositivo sin necesidad de WhatsApp Web tradicional.

Este proyecto está diseñado para ser:

- ⚡ Rápido
- 🔧 Personalizable
- 🧠 Escalable
- 🔥 Fácil de usar

---

✨ Características

📥 Descargas

- YouTube (audio/video)
- TikTok sin marca de agua
- Instagram media downloader

🎨 Stickers

- Crear stickers desde imágenes
- Convertir GIF/video a sticker
- Stickers personalizados

🤖 Inteligencia Artificial (opcional)

- Chat con IA
- Generación de texto
- Asistencia automática

👥 Grupos

- Menciones masivas
- Administración básica
- Herramientas para admins

🎮 Diversión

- Juegos simples
- Frases aleatorias
- Comandos interactivos

⚙️ Configuración

- Prefijo personalizable
- Nombre del bot editable
- Menú dinámico con imagen/video

🚀 Instalación

📱 Requisitos

- Node.js 18 o superior
- Git
- Termux (Android) o VPS

---

⚡ Instalación rápida

pkg update && pkg upgrade
pkg install nodejs git

git clone https://github.com/anyelmercado52-ux/hiro-bot-MD-bot
cd hiro-bot-MD

npm install
npm start

---

📲 Vinculación (Pairing Code)

1. Ejecuta el bot:

npm start

2. Escribe tu número de WhatsApp (ej: "1809XXXXXXX")

3. Recibirás un código:

XXXX-XXXX

4. Ve a WhatsApp:

- Dispositivos vinculados
- Vincular con número
- Ingresa el código

---

🔐 Seguridad

⚠️ Importante:

No compartas estos archivos:

- "/auth"
- "/session"
- ".env"

Agrega esto a tu ".gitignore":

node_modules/
auth/
session/
.env
